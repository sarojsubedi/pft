# Personal Finance Tracker
A simple mockup of a personal finance tracker built while learning React. This project is currently a work in progress and serves as a practice ground for React fundamentals.

## Features

- **Simple UI Mockup**: A basic interface to visualize financial data.
- **Work in Progress**: Actively adding new features as I learn more about the React ecosystem.

## Getting Started

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/sarojsubedi/pft.git
   ```
2. Navigate to the project directory:
   ```bash
   cd pft
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

To start the application, run:
```bash
npm run dev
```
or
```bash
npm run build
```
